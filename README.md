# Battery Report Viewer by Kamil Haroon

A lightweight Windows tool that triggers the system-generated battery report and displays it in a Browser.

## Features
- Generates Windows battery report using `powercfg /batteryreport`
- Shows battery health, full charge capacity, design capacity, cycle count, and more
- Opens automatically in your browser
- Portable â€” no installation needed

## Usage
1. Download the EXE from the Releases page
2. Run it
3. Your browser opens the battery report
